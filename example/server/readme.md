# Example gRPC server and client

## Usage
1. deploy server locally
```sh
nvm use || nvm install
npm i -P
npm run build-server
npm run start-server
```

2. develop server locally
```sh
nvm use || nvm install
npm i
npm run dev-server
```

3. deploy client locally
```sh
nvm use || nvm install
npm i -P
npm run build-client
npm run start-client
```

4. develop client locally
```sh
nvm use || nvm install
npm i
npm run dev-client
```

5. build proto
```sh
nvm use || nvm install
npm i
npm run build-proto
```
